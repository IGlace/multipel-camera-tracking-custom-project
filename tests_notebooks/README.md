# Experiment Notebooks

This folder contains Jupyter notebooks used to manually experiment with and validate individual components of the MCMT platform.

The notebooks are meant for:
- component-level inspection
- debugging
- visual validation
- quick research experiments

They do not replace formal automated tests.

Planned notebook families include:
- detection
- segmentation
- ReID
- attributes
- graph construction and visualization
- spatial graph and temporal graph single-epoch experiments
- frame-by-frame and tracklet tracking
- graph and non-graph modes
- realtime modes
- output sinks and MOT export
